# 手写错题分析 — 样本说明

5 份原始手写样本，按风格特征分类：

## `听力1.txt` — 最丰富的一份

特点：
- 错因口语化典范（"纯蒙的，其实我会"、"只听到了Italian，后面具体是什么没注意没听到"）
- 生词直接贴字典（`简明释义` + `网络`）
- 选项分析 ABCD 全覆盖（这道比较长）
- 翻译是逐句意译
- 多道题混排（1, 13, 20, 23, 25, 26, M2 的 4, 7, 15）
- 有"网络反感 /厌恶 /讨厌的事"这种原貌从欧路复制的感觉

值得复制的特征：错因的第一人称暴露、自嘲、原貌字典

## `听力2.txt` — 短小典型

特点：
- 每道题 2-4 段
- 生词贴字典 + 偶尔留"全部网络翻译"
- 错因更短："没听到 celebrate, 只听到和谁什么结束论题"

值得复制的特征：紧凑、不啰嗦

## `阅读1.txt` — 学术阅读一气呵成

特点：
- 13 题整道题"一气呵成"地推理：关键词 → 新发现 → 推断 → 选项
- 没"框架/主旨/细节"那种分类
- 选项只写正确的那个

值得复制的特征：学术阅读短题不用硬套结构

## `阅读2.txt` — 翻译 + 生词原貌

特点：
- 翻译完整
- 生词带音标 + 简明释义 + 网络翻译
- 错因有上下文（"大概看了一眼"）

值得复制的特征：字典原貌三栏

## `阅读3.txt` — 学术阅读 + 复杂分析

特点：
- 14 题用箭头 → 引出选项判断
- 16 题用"写作意图/修辞目的"切入
- 9 题用"根本区别"做对比
- 10 题 / 13 题：生词整理（带音标 + 简明释义 + 网络）

值得复制的特征：箭头 + 简评的轻量分析方式

---

## 共同的"人味"元素

5 份样本都有的：
1. 标题裸 `M1` `M2` 不用框线
2. 题号 `1）` `13）` 中文括号
3. 错因第一句话就是"我..."，不是客观描述
4. 生词贴字典原貌（三栏）
5. 翻译逐句意译
6. 偶尔留"？" / "其实我会" / 自我吐槽
7. 不用"重点改进" "错因归类" 结构化总结
